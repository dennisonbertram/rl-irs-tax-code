# Current Intent

## What
Fine-tune a small LLM (Qwen 2.5 3B) on the IRS Internal Revenue Code using supervised fine-tuning (SFT) followed by reinforcement learning (DPO/GRPO), then deploy via Ollama.

## Why
Test/demonstration of local post-training and RL capabilities on domain-specific legal text.

## Key Constraints
- 41GB free disk space
- Apple M4 Max with 128GB unified memory
- Must run entirely locally
- Final model must be usable through Ollama
