# Project: IRS Tax Code RL Training

## GPT-5.4 Usage
When complex decisions arise (architecture, training hyperparameters, data pipeline design, evaluation strategy), consult GPT-5.4 on extra high reasoning via the ralph loop or direct consultation. Use it for:
- Training pipeline architecture decisions
- Hyperparameter selection and optimization strategy
- Data quality and preprocessing validation
- Evaluation methodology design
- Debugging training instabilities

## Project Context
This project fine-tunes a small local LLM (via Ollama) on the IRS tax code using reinforcement learning and supervised fine-tuning techniques.
